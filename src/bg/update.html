在线词典助手-升级说明

您已升级至在线词典助手 0.9.9 版本。如需完整文档或插件最新信息，请访问 项目主页

0.9.9 版更新
  1. 新增：网站规则（Site Rules）功能。支持针对不同域名设置个性化的取词配置（如特定词典、Anki 牌组、最大词数限制等）。
  2. 新增：火狐云同步 (Firefox Sync) 支持。网站规则现已接入混合存储策略，可跨设备自动同步您的域名个性化配置。
  3. 新增：支持网站规则的导入与导出 (JSON)。方便用户快速备份或在不同环境间共享特定网站的翻译规则。
  4. 修复：解决了网站规则编辑器与弹窗修改无法同步的问题。
  5. 优化：重新设计了选项页布局，确保在不同配置状态下界面始终美观、整齐。
0.9.8 版更新
  1. 修复：弹窗内使用鼠标滚轮时单次滚动幅度过大（跳步）的问题。
  2. 修复：划词探测器逻辑错误，导致取词时错误包含单词末尾标点（如 Example: 误取包含冒号）的 bug，重构后取词更加精准。
0.9.7 版更新
  1. 新增：支持暗色模式/深色主题，智能反转硬编码词典颜色保护视力
  2. 新增：Ankiconnect 制卡操作加入 Toast 弹窗反馈，提供详细成功/重复/失败提示（可关闭）
  3. 新增：可设置最大选词数量，包含针对中文/英文混合分词的精确字符数计算
  4. 新增：非目标语言划词时智能跳过查询，避免无效报错
  5. 修复：在 Google Docs 等各种 contenteditable 页面中误触发 Shift 取词的问题
  6. 优化：划词性能、音频播放防泄漏与鼠标重操作节流，提升扩展稳定与流畅性
0.9.6 版更新
  1. 新增：支持自定义 Ankiconnect 连接地址（IP/端口），方便局域网或远程 Anki 连接
  2. 修复：鼠标滚轮在查词弹窗中直接跳到顶部或底部的滚动异常问题
  3. 修复：ankiweb.js 中 Promise.reject 调用错误导致控制台报错的问题
  4. 修复：Android 版火狐浏览器上 chrome.commands 未定义导致扩展无法初始化的问题
  5. 修复：Android 版火狐浏览器上 Ankiconnect URL 输入框无法显示的问题

Online Dictionary Helper -- Upgrade Notes

Your extension was upgraded to v0.9.9. If you need more details, please visit Project Home

version 0.9.9
  1. New: Site Rules feature. Allows domain-specific configuration overrides (Dictionary, Max words, Anki Deck/Note types, etc.).
  2. New: Firefox Sync support. Site Rules are now automatically synchronized across all your devices using Firefox accounts.
  3. New: Import/Export (JSON) for Site Rules. Easily backup or share your domain-specific translation settings.
  4. Fix: Synchronized the data flow between Site Rule editor and Popup quick-settings.
  5. Optimize: Redesigned the Options UI for better consistency and responsiveness across all tabs.
version 0.9.8
  1. Fix: Mouse wheel scroll step size was too large in the popup window.
  2. Fix: Text selection boundary detector bug that incorrectly included ending punctuation (e.g. colon in Example:). Rewritten for perfect precision.
version 0.9.7
  1. New: Dark mode support with intelligent color inversion for hardcoded dictionary styles
  2. New: Anki built-in Toast notifications for showing detailed success/duplicate/error states
  3. New: Maximum selected words limit with accurate CJK/Latin word counting
  4. New: Smart language filtering to skip dictionary lookups for non-matching languages
  5. Fix: Prevent Shift key hotkey from triggering in contenteditable elements like Google Docs
  6. Optimize: Improved frontend performance with better throttling and audio memory cache
version 0.9.6
  1. New: Support custom Ankiconnect URL (IP/port) for LAN or remote Anki connections
  2. Fix: Mouse wheel in the popup window jumping to top/bottom instead of smooth scrolling
  3. Fix: Incorrect Promise.reject usage in ankiweb.js causing unhandled rejection errors
  4. Fix: chrome.commands undefined on Android Firefox causing extension initialization failure
  5. Fix: Ankiconnect URL input field not showing on Android Firefox
