# 更新日志 (Changelog)

## 1.0.0
- 新增：**最近添加卡片**查看功能。在弹窗 (popup.html) 中可直接查看最近添加的 5 个单词，并支持点击一键删除。
- 优化：**Anki 制卡提示**增强。成功添加卡片后，若 Anki 中已存在同名卡片，会在 Toast 提示中显示具体重复数量。
- 修复：重构了词典语言检测逻辑。修复了类似 `general_Makenotes` 等非标准命名词典导致的取词拦截 Bug。

## 0.9.9
- 新增：**网站规则（Site Rules）**功能。支持针对不同域名设置个性化的取词配置（如特定词典、Anki 牌组、最大词数限制等）。
- 新增：火狐云同步 (Firefox Sync) 支持。网站规则现已接入混合存储策略，可跨设备自动同步您的域名个性化配置。
- 新增：支持网站规则的导入与导出 (JSON)。方便用户快速备份或在不同环境间共享特定网站的翻译规则。
- 修复：解决了网站规则编辑器与弹窗修改无法同步的问题。
- 优化：重新设计了选项页布局，确保在不同配置状态下界面始终美观、整齐。

## 0.9.8
- 修复：弹窗内使用鼠标滚轮时单次滚动幅度过大（跳步）的问题。
- 修复：划词探测器逻辑错误，导致取词时错误包含单词末尾标点（如 `Example:` 误取包含冒号）的 bug，重构后取词更加精准。

## 0.9.7
- 新增：支持**暗色模式/深色主题**，智能反转硬编码词典颜色保护视力。
- 新增：Ankiconnect 制卡操作加入**Toast弹窗反馈**，提供详细成功/重复/失败提示（可关闭）。
- 新增：可设置**最大选词数量**，包含针对中文/英文混合分词的精确字符数计算。
- 新增：非目标语言划词时智能跳过查询，避免无效报错。
- 修复：在 Google Docs 等各种 contenteditable 页面中误触发 Shift 取词的问题。
- 优化：划词性能、音频播放防泄漏与鼠标重操作节流，提升扩展稳定与流畅性。

## 0.9.6
- 新增：支持**自定义 Ankiconnect 连接地址**（IP/端口），方便局域网或远程 Anki 连接。
- 修复：鼠标滚轮在查词弹窗中直接跳到顶部或底部的滚动异常问题。
- 修复：ankiweb.js 中 Promise.reject 调用错误导致控制台报错的问题。
- 修复：Android 版火狐浏览器上 chrome.commands 未定义导致扩展无法初始化的问题。
- 修复：Android 版火狐浏览器上 Ankiconnect URL 输入框无法显示的问题。
